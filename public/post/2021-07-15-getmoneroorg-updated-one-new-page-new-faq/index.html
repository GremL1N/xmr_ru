Это последний отчёт, публикуемый в рамках моего предыдущего предложения, внесённого через CCS, и в нём отражены все последние изменения Getmonero примерно за последние 30 дней. Мне также хотелось бы поблагодарить всех тех, кто поддержал или внёс свой вклад по моему последнему предложению, опубликованному через CCS, которое теперь получает необходимое финансирование и, по сути, уже практически полностью профинансировано. Спасибо :)

Обновление Getmonero

Это основные изменения на Getmonero.org с момента публикации моего последнего отчёта:

  • На домашней странице появилась новая верхняя панель с новым слоганом и новыми командными кнопками - #1633
  • Добавлена новая страница Tools (Инструменты), заменяющая собой соответствующее устаревшее руководство Tools - #1670
  • В раздел FAQ добавлен новый пункт: Are there known vulnerabilities in Monero? (Существуют ли известные уязвимости Monero?) - #1667
  • Добавлен новый пост: General Fund Report (Отчёт по общему фонду) - #1686
  • В раздел Library (Библиотека) добавлена новая «шпаргалка» по RingCT - #1655
  • Добавление переводов сайта на 12 языках от 30 переводчиков (спасибо!) - #1664
  • Обновлён стиль страницы майнинга - #1656
  • Обновлено руководство пользователя [Getting started with the CLI wallet (Начало работы с CLI-кошельком)](Getmonero.org updated: One new page, new FAQ, translations, revamps and more) - #1661, #1665
  • Удалены ссылки сервиса MatterMost (который был заменён на Matrix) - #1662
  • В разделах Workgroups, Hangouts и readme произведена замена Freenode на Libera - #1652, #1677, #1693
  • В разделе FAQ в пункт Is it true that Monero has a hardfork every 6 months? (Правда ли, что хардфорки Monero происходят каждые 6 месяцев?) добавлена дата последнего хардфорка - #1674
  • В разделе Encryption (Шифрование) Moneropedia удалены все упоминания Kovri - #1653
  • Добавлен шаблон запроса для удаления предпринимателей из соответствующего списка - #1673
  • В дорожной карте упорядочены уже реализуемые и будущие задачи - #1679
  • В блог внесены минимальные исправления и корректировки - #1689
  • В элемент img добавлено правое поле для внешних ссылок - #1684
  • Оптимизировано смещение раскладки на домашней странице - #1678
  • Обновлены языковые файлы в руководстве пользователя Verify binaries on Linux, Mac, or Windows command line (advanced) (Верификация двоичных файлов в командной строке Linux, Mac или Windows (продвинутый уровень)) - #1639
  • Добавлены хеши исходного кода (архив) в подраздел verify (верификация) раздела Downloads (Для скачивания) - #1691

Данный список не включает в себя всех изменений, внесённых в разделы веб-сайта или в репозиторий. Чтобы получить более подробную информацию, ознакомьтесь со списком всех коммитов или c историей GitHub в течение последнего месяца.
