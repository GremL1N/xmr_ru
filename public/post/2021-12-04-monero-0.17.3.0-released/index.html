Доработанная промежуточная версия Monero 0.17.3.0 Oxygen Orion

Это доработанная промежуточная версия v0.17.3.0 программного обеспечения Monero. Эта версия добавляет поддержку P2Pool.

2021-12-04
Новости

Обзор

Вот некоторые особенности этой промежуточной версии:

  • Добавлена поддержка P2Pool (#7964)
  • Демон: добавлена поддержка конфигурации socks5 прокси по средствам команды --proxy (#7326, #7616)
  • Демон: исправлен ползунок статуса синхронизации (#7873)
  • Демон: исправлено ложное отклонение загруженных блоков (#8022)
  • Демон: отключено ограничение RPC для получения выходных данных для предварительных rct выводов (исправлен DoS) (#8084)
  • Демон: добавлены начальные узлы (#7664, #7753)
  • Кошелёк: исправлено шифрование ключа при смене пароля с 0/1 на 2 (#8014)
  • Кошелёк: исправлена точность выбора приманок (#7798)
  • Кошелёк: уменьшено окно недавних трат при выборе приманок (#7993)
  • Кошелёк: исправлена ошибка обрезания интегрированных адресов при экспорте истории в CSV (#7961)
  • Кошелёк: добавлено ограничение RPC вызовов в функции get_outs.bin (#7796)
  • Кошелёк: добавлены удобочитаемые сообщения об ошибках при работе с Ledger (#8039)
  • RPC: исправлена ошибка get_transactions (#7959)
  • RPC: добавлен новый calcpow метод (#8075)
  • Добавлена поддержка подстановочных знаков CORS (#7952)
  • Исправлена ошибка журнала (#7929)
  • Mac: исправлена компиляция на ARM (#7435)
  • Mac: исправлена ошибка медленного хэша в стеке для ARM64 (#8032)
  • LMDB: исправлена тупиковая ситуация при обнаружении изменения размера (#7958)
  • Добавлена динамическая поддержка AES для двоичных файлов ARMv8 на Raspberry Pi (#8005)
  • Обновление зависимостей: OpenSSL, RandomX, unwind, hidapi, zeromq (#7933, #8002, #8037, #8049, #8056, #8072)
  • Исправлены прочие незначительные ошибки

Полный список изменений выложен на GitHub вместе с исходным кодом.
