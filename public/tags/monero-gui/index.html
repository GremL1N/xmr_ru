Доработанная версия Monero GUI 0.17.3.0 Oxygen Orion

Это доработанная промежуточная версия v0.17.3.0 программного обеспечения Monero GUI

2021-12-04
Новости
Обзор Описание последней версии CLI содержится в предыдущем посте. Вот некоторые особенности этой промежуточной версии: Добавлена функция импорта / экспорта выходов (#3435) Добавлена функция экспорта всех образов ключей (#3282) Обновление переводов (#3773) Добавлена возможность оставить комментарий при экспорте CSV файла (#3572) Добавление цветовой шкалы надежности задаваемого пароля (#3682) Добавлены функциональные кнопки Вырезать, Копировать, Удалить и Выбрать все в контекстное меню кошелька(#3689) Исправлено переполнение буфера в библиотеке zxcvbn (#3735) Исправлена проблема со шрифтами в Arch Linux (#3753) Исправлена компиляция для Apple ARM (#3697, #3723, #3731) Переработка дизайна страницы Майнинг (#3742) Полный список изменений выложен на GitHub вместе с исходным кодом.
